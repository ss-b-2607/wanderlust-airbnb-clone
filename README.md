# Wanderlust Airbnb Clone

A full-stack Airbnb-style listing website built using Node.js, Express, MongoDB, and EJS.

# Features
- Create new listings
- Edit listings
- Delete listings
- Image URL support
- Error handling

# Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
